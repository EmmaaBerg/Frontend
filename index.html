Frontend

What is frontend?

Evertime you visit a website on your desktop or phone you are exposed to frontend. Frontend is what you as a user can see and interact with on websites or web applications.

Example of frontend elemets are

  • text
  • colors
  • images/videos
  • buttons
  • forms

Read more about frontend here or check out this introduction video

Frontend development

Web languages like HTML, CSS and JavaScript are used in frontend development to create the layout with the goal to make it a user friendly experience.

One challenge with frontend development today is that users have many different types of devices with various screen sizes and operating systems. Therefore a frontend developer have to make sure a website or application works on different devices and operating systems likewise different browsers.
